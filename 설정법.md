Zhipu AI 및 Z.ai 5시간 토큰 사용 한도 현황을 확인하세요.
MCP 월별 할당량 의 의미와 재설정 규칙을 이해하세요.
할당량 출력에서 ​​진행률 표시줄, 사용량, 합계 등의 정보를 읽어옵니다.
사용량 경고가 발생하는 시점을 파악하세요.
현재 당면 과제
Zhipu AI 또는 Z.ai를 사용하여 애플리케이션을 개발하지만 다음과 같은 문제에 자주 직면합니다.

5시간 토큰 사용 한도 에 얼마나 남았는지 확실하지 않습니다.
요청 한도를 초과한 후 요청이 실패하여 개발 진행에 영향을 미칩니다.
MCP 월별 할당량 의 구체적인 의미가 불분명합니다.
할당량을 확인하려면 두 플랫폼에 각각 로그인해야 하므로 불편합니다.
이 기능을 언제 사용해야 할까요?
당신이:

Zhipu AI/Z.ai API를 사용하여 애플리케이션을 개발하세요.
토큰 사용량을 모니터링하여 제한을 초과하지 않도록 해야 합니다.
MCP 검색 기능의 월별 사용량 제한에 대해 알고 싶습니다.
Zhipu AI와 Z.ai를 모두 사용 중이며 통합된 할당량 관리를 원합니다.
핵심 개념
Zhipu AI 와 Z.ai의 할당량 시스템은 두 가지 유형으로 나뉩니다.

할당 유형	의미	리셋 사이클
토큰 사용 시간 제한 5시간	API 요청에 대한 토큰 사용 제한	5시간마다 자동으로 재설정됩니다.
MCP 월별 할당량	MCP(모델 컨텍스트 프로토콜) 검색 횟수에 대한 월별 제한	매달 초기화됩니다
이 플러그인은 공식 API를 호출하여 해당 데이터를 실시간으로 조회하고 진행률 표시줄 과 백분율을 통해 남은 할당량을 시각적으로 표시합니다 .

MCP란 무엇인가요?

MCP (모델 컨텍스트 프로토콜)는 Zhipu AI에서 제공하는 모델 컨텍스트 프로토콜로, AI 모델이 외부 리소스를 검색하고 참조할 수 있도록 합니다. MCP 월별 할당량은 매달 사용 가능한 검색 횟수를 제한합니다.
